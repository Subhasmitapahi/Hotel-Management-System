# Hotel-Management-System
A desktop-based Hotel Management System developed using Core Java, Swing (GUI), JDBC, and MySQL, designed to manage hotel operations such as customer handling, room management, and booking processes. The application follows a layered architecture with DAO design pattern to ensure clean separation of concerns and maintainable code.
